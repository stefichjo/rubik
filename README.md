# Notation for Rubik Algorthims

* `(A₁ .. A₂)` ≡ `A₁A₂`
* `(A₁ .. A₂): B` ≡ `A₁BA₂`
* `(A₁ .. A₂) (B₁ .. B₂)` ≡ `(A₁B₁ .. A₂B₂)` ≡ `A₁B₁A₂B₂`

* `A2` ≡ `A²`

|-|-|-|-|-|-|
| `(AB)²` | `(A: B)²` | `(A)²: B` | `[AB]`   | `[A: B]` | `[A]: B` |

## Format

### `,`

`<A>, <B>` ≡ `<A, B>`
<br>≡
<br>`<A B>`
<br>≡
<br>`<A> <B>`

### `;`

`<A>; <B>` ≡ `<A; B>`
<br>≡
<br>`<A>`
<br>`<B>`
<br>≡
<br>`<A`
<br>&nbsp;&nbsp;&nbsp;`B>`

### `:`

`<A>: <B>` ≡ `<A: B>`
<br>≡
<br>`<A>`
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;`<B>`
<br>≡
<br>`<A`
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;`B>`

# Traditional Notation

* Commutator: `ABA'B'` ≡ `[A, B]`
* Conjugate: `ABA'` ≡ `[A: B]`
